# 🐍 Python - Exercícios e Projetos de Aula

Repositório criado para versionamento dos exercícios e projetos desenvolvidos
durante as aulas de **Introdução à Programação com Python**, utilizando Git e
GitHub para controle de versão, organização e backup do código.

## 🚀 Como executar

**Pré-requisitos:** Python 3.x e Git instalados.

```bash
git clone https://github.com/GuiKadz/python-faculdade-tde-1.git
cd python-faculdade-tde-1
python aula03-introducao/exercicios.py
```

## 📝 Padrão de commits (Conventional Commits)

- `feat:` — novo exercício adicionado
- `fix:` — correção de exercício
- `docs:` — alterações na documentação
- `refactor:` — refatoração sem mudança de comportamento
